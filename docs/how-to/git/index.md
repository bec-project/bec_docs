# Working With Git

This section contains Git-focused how-to guides for working with BEC repositories. These guides are intended for users who are responsible for maintaining BEC at the beamline but are not necessarily familiar with Git. The guides will walk you through the process of making changes to your plugin repository and merging them into the main branch.


## Overview

It covers the following topics:

- [Add Changes to Your Plugin Repository](./add-changes-to-plugin-repository.md): How to add changes to your plugin repository.
- [Merge Changes to `main`](./merge-changes-to-main.md): How to merge a branch into the main branch of your plugin repository.
