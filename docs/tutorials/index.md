# Tutorials

Tutorials help a reader learn by doing.

## Goals

- Guided walkthroughs for specific tasks or features
- Assume the reader has no prior knowledge of the topic or BEC

## Audience

- New users
- Users who have not used a specific feature in a while

## Content

The initial tutorial track is PSI-specific and centered on a getting-started path:

- launch BEC
- add a demo config
- add a device to the config through the GUI
- modify the properties of a device through the CLI
- run a scan through the GUI and the CLI
- move a motor through the GUI and the CLI
- inspect the scan history through the CLI
