# Documentation Model

This repository uses the Diataxis framework as its organizing principle.

## Why Diataxis

The existing source material spans several repositories and several writing styles. Diataxis gives the combined documentation a stable editorial model that helps us merge that material without turning the site into a repository index.

## What belongs where

- Tutorials teach through a guided path.
- How-to guides solve a concrete task.
- Reference documents exact facts and interfaces.
- Explanation builds understanding and context.

## Editorial rule

When content is brought in from `bec`, `bec_widgets`, or `ophyd_devices`, it should be rewritten or reshaped to match the Diataxis category it belongs to instead of preserving its original structure by default.
