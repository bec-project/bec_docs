# References

Use this section when you need precise technical information about BEC components, APIs, and structures.
Reference material is designed for lookup: it describes things as they are, so you can check names, interfaces,
options, and behavior quickly.

Unlike tutorials or how-to guides, reference pages are not primarily written to teach or guide you through a task.
They are here to give you reliable facts in a form you can return to whenever you need them.

## How to use this section

Reference pages are usually not meant to be read from beginning to end. In most cases, you will arrive here by
following links from examples, how-to guides, or learning material when you need to check a specific technical detail.
